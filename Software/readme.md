# Bootloader
This is the bootloader for wueHans. It will load the first .elf file that it can find on the SD-Card in the top directory. Check https://github.com/Byter64/Hans2/wiki/Building-wueHans to see how it is built.

# CowShooter
CowShooter is a demo game for wueHans. It is not integrated into the cmake structure of wueHans and contains its own. It uses the [Hans2SoftwareTemplate](https://github.com/Byter64/Hans2SoftwareTemplate). Check this one out, if you want to know, how to build CowShooter.

Gameplay can be found here: https://www.youtube.com/watch?v=B-wMZS0atJ4
